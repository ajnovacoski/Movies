# Movies
## Atividade de Tópicos Especiais em Ciência da Computação

### Usando ionic, typescript, fazer um app que tenha um CRUD do elemento "Filme".
 Nesse caso, temos o Filme, com atributos: 
  -  Título
  -  Ano de lançamento
  -  Sinopse
  -  Diretor
  -  Duração(em minutos)
  -  Classificação Indicativa
  -  Gênero
  -  Orçamento.
  -  **Capa(Imagem)**
